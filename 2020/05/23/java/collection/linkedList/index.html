LinkedList底层用双向链表实现的存储。特点：查询效率低，增删效率高，线程不安全。

​ 双向链表也叫双链表，是链表的一种，它的每个数据节点中都有两个指针，分别指向前一个节点和后一个节点。 所以，从双向链表中的任意一个节点开始，都可以很方便地找到所有节点。
